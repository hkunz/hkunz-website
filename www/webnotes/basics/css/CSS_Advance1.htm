CSS Dimension

The CSS dimension properties allow you to control the height and width of an element. It also allows you to increase the space between two lines.

  • width/height: auto/30px/50%;
  • max-width/height: 10px/20%; - ex.setting the width and height(margin) of text
  • min-width: 1000px/200%; - ex.for text so that it has a nowrap-like effect
  • line-height: 2/30px/90%; - can be set through number value or pixel value or percent value

CSS Positioning

The CSS positioning properties allow you to specify the left, right, top, and bottom position of an element. It also allows you to set the shape of an element, place an element behind another, and to specify what should happen when an element's content is too big to fit in a specified area.

  • position:relative;left:-20px;
  • position:absolute;left:100px;top:150px;
  • position:fixed;top:30px;right:5px;
  • position:absolute;clip:rect(0px 50px 200px 0px); - use it to cut images
  • background-color:#00FFFF;width:150px;height:150px;overflow: scroll/hidden/auto
  • This uses vertical-align:text-bottom; to place image in paragraph. Other values: text-bottom/text/top.
  • position:absolute;left:0px;top:0px;z-index:-1; (-1,[0-default],1) Z-index can be used to place an element "behind" another element.

CSS Pseudo-Classes

CSS pseudo-classes are used to add special effects to some selectors.

Syntax

The syntax of pseudo-classes: selector:pseudo-class {property: value}

CSS classes can also be used with pseudo-classes: selector.class:pseudo-class {property: value}

Anchor Pseudo-Classes

  • a:link {color: #FF0000} /* unvisited link */
  • a:visited {color: #00FF00} /* visited link */
  • a:hover {color: #FF00FF} /* mouse over link */
  • a:active {color: #0000FF} /* selected link */
